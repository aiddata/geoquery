# Contributing

We welcome contributions to GeoQuery — whether that's reporting bugs, suggesting new datasets, improving documentation, or submitting code changes.

## Reporting issues

Please open an issue on [GitHub](https://github.com/aiddata/geoquery/issues) for:

- Bug reports
- Feature requests
- Dataset requests
- Documentation improvements

## Contributing code

1. Fork the repository on [GitHub](https://github.com/aiddata/geoquery)
2. Create a feature branch from `main`
3. Make your changes and ensure existing tests pass
4. Submit a pull request with a clear description of the change

For larger changes, open an issue first to discuss the approach before investing significant time.

## Contributing data or boundaries

If you have a dataset or boundary collection you'd like to see added to GeoQuery, please email us at [geo@aiddata.wm.edu](mailto:geo@aiddata.wm.edu) with a description of the data, its source, and any relevant licensing information.

## Contact

For any other questions, reach us at [geo@aiddata.wm.edu](mailto:geo@aiddata.wm.edu).

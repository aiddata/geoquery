# Why Kubernetes?

Kubernetes allows us to build infrastructure for GeoQuery that can be quickly spun up and down on different clusters.
Additionally, Kubernetes provides helpful resource abstractions like jobs and services that allows us to interface various aspects of GeoQuery in standard ways.

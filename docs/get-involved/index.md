# Get Involved

GeoQuery is a collaborative platform built and maintained by [AidData](https://www.aiddata.org) at William & Mary. There are several ways to get involved.

## Contributing

Help improve GeoQuery by contributing code, documentation, or data. See the [Contributing](contributing/index.md) section for guidelines on how to get started.

## Partner with Us

Interested in a research partnership or contributing datasets? Learn more about [partnering with us](partner-with-us.md).

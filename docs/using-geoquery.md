# Using GeoQuery

## Submitting a request

1. Open the GeoQuery application and use the map to select your area of interest
2. Browse and select the datasets you want data extracted for
3. Review your selection, then click **Submit Request** in the upper right
4. Enter your email address and click the **Submit** button
5. You will receive an email when your results are ready

## Boundary files

GeoQuery supports both built-in administrative boundaries and custom boundary files.

To use your own boundary, use the custom boundary upload form available from the main interface. Boundaries should be provided as shapefiles.

## Multiple boundaries

At this time, GeoQuery processes one boundary per request. To get data for multiple regions, submit a separate request for each. Previously processed requests are cached, so repeat selections are returned much faster.

## Data formats

Each completed request includes:

| File | Description |
|------|-------------|
| CSV | Extracted values for use in Excel, Stata, SPSS, or similar tools |
| Shapefile | Boundary geometry for use in GIS software |
| PDF / HTML | Metadata including dataset descriptions, sources, and citations |

## Turnaround time

Many requests return almost immediately thanks to preprocessing and caching. Requests that require new processing — particularly those covering large areas or many project locations — may take up to several hours.
